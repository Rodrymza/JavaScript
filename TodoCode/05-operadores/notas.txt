Operadores de asignacion

= // Asignación simple, asigna el valor del operando derecho al operando izquierdo
+= // Asignación de suma, suma el valor del operando derecho al operando izquierdo y asigna el resultado
-= // Asignación de resta, resta el valor del operando derecho del operando izquierdo y asigna el resultado
*= // Asignación de multiplicación, multiplica el valor del operando derecho por el operando izquierdo y asigna el resultado
/= // Asignación de división, divide el valor del operando izquierdo por el operando derecho y asigna el resultado
%= // Asignación de módulo, calcula el resto de la división del operando izquierdo por el operando derecho y asigna el resultado
**= // Asignación de exponenciación, eleva el operando izquierdo a la potencia del operando derecho y asigna el resultado
<<= // Asignación de desplazamiento a la izquierda, desplaza los bits del operando izquierdo hacia la izquierda según el valor del operando derecho y asigna el resultado
>>= // Asignación de desplazamiento a la derecha, desplaza los bits del operando izquierdo hacia la derecha según el valor del operando derecho y asigna el resultado
>>>= // Asignación de desplazamiento a la derecha sin signo, desplaza los bits del operando izquierdo hacia la derecha según el valor del operando derecho sin considerar el signo y asigna el resultado
&= // Asignación de AND bit a bit, realiza una operación AND bit a bit entre el operando izquierdo y el operando derecho y asigna el resultado
^= // Asignación de XOR bit a bit, realiza una operación XOR bit a bit entre el operando izquierdo y el operando derecho y asigna el resultado
|= // Asignación de OR bit a bit, realiza una operación OR bit a bit entre el operando izquierdo y el operando derecho y asigna el resultado

Operadores aritmeticos

+ // Suma, devuelve la suma de los operandos
- // Resta, devuelve la diferencia entre los operandos
* // Multiplicación, devuelve el producto de los operandos
/ // División, devuelve el cociente de los operandos
% // Módulo, devuelve el resto de la división de los operandos
** // Exponenciación, devuelve el resultado de elevar el operando izquierdo a la potencia del operando derecho
++ // Incremento (+1)
-- // Decremento (-1)

Operadores de comparacion

== // Igualdad, devuelve True si los operandos son iguales
!= // Desigualdad, devuelve True si los operandos no son iguales
> // Mayor que, devuelve True si el operando izquierdo es mayor que el operando derecho
< // Menor que, devuelve True si el operando izquierdo es menor que el operando derecho
>= // Mayor o igual que, devuelve True si el operando izquierdo es mayor o igual que el operando derecho
<= // Menor o igual que, devuelve True si el operando izquierdo es menor o igual que el operando derecho
=== // Identidad, devuelve True si los operandos son idénticos
!== // No identidad, devuelve True si los operandos no son idénticos

Operadores condicionales


?: // Condicional ternario, devuelve el valor del operando izquierdo si la condición es verdadera, de lo contrario devuelve el valor del operando derecho

&& // Condicional AND, devuelve True si ambos operandos son verdaderos
|| // Condicional OR, devuelve True si alguno de los operandos es verdadero
! // NOT, devuelve True si el operando es falso, de lo contrario devuelve False
