Función Prompt de JavaScript

La función prompt() es una función de JavaScript que se utiliza para solicitar una entrada de texto al usuario.

Esta función muestra un cuadro de diálogo con un mensaje y un campo de texto para que el usuario ingrese su respuesta.

La función prompt() devuelve el valor ingresado por el usuario como una cadena de texto.

Sintaxis: prompt(mensaje, valorPorDefecto)

Parámetros:

  • mensaje: el mensaje que se mostrará en el cuadro de diálogo.
  • valorPorDefecto: el valor que se mostrará en el campo de texto por defecto.

Ejemplo:

  
    let nombre = prompt("Ingrese su nombre:");
    console.log("Hola, " + nombre);

La función alert() es una función de JavaScript que se utiliza para mostrar un mensaje al usuario.

Esta función muestra un cuadro de diálogo con un mensaje y un botón de "Aceptar" para que el usuario lo cierre.

La función alert() no devuelve ningún valor.

Sintaxis: alert(mensaje)

Parámetros:

  • mensaje: el mensaje que se mostrará en el cuadro de diálogo.

Ejemplo:

  
    alert("Hola, mundo!");

La función confirm() es una función de JavaScript que se utiliza para mostrar un mensaje al usuario y solicitar su confirmación.

Esta función muestra un cuadro de diálogo con un mensaje y dos botones: "Aceptar" y "Cancelar".

La función confirm() devuelve un valor booleano que indica si el usuario aceptó o canceló la solicitud.

Sintaxis: confirm(mensaje)

Parámetros:

  • mensaje: el mensaje que se mostrará en el cuadro de diálogo.

Ejemplo:

  
    let respuesta = confirm("¿Estás seguro de que deseas continuar?");
    if (respuesta) { console.log("Siguiendo") 
} else { console.log("Cancelando") }
