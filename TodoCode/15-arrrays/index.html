Arreglos en JavaScript

Un arreglo es una colección de elementos de cualquier tipo, incluyendo números, cadenas de texto, objetos y otros arreglos.

Se pueden crear de varias maneras:

// Declaración literal
let frutas = ['manzana', 'plátano', 'naranja'];

// Declaración con constructor
let colores = new Array('rojo', 'verde', 'azul');

Los arreglos tienen métodos y propiedades que permiten manipularlos, como:

  • length: devuelve el número de elementos en el arreglo.
  • push(): agrega un elemento al final del arreglo.
  • pop(): elimina el último elemento del arreglo.
  • indexOf(): devuelve el índice del primer elemento que coincide con el valor especificado.
// Declaración literal
let frutas = ['manzana', 'plátano', 'naranja'];

// Declaración con constructor
let colores = new Array('rojo', 'verde', 'azul');

Los arreglos tienen métodos y propiedades que permiten manipularlos, como:

  • length: devuelve el número de elementos en el arreglo.
  • push(): agrega un elemento al final del arreglo.
  • pop(): elimina el último elemento del arreglo.
  • indexOf(): devuelve el índice del primer elemento que coincide con el valor especificado.
  • includes(): devuelve un booleano que indica si el arreglo incluye un elemento determinado.
  • join(): devuelve una cadena de texto que une todos los elementos del arreglo.
  • slice(): devuelve un nuevo arreglo que es una copia de una parte del arreglo original.
  • splice(): elimina o reemplaza elementos en el arreglo.
  • forEach(): ejecuta una función para cada elemento del arreglo.
  • map(): devuelve un nuevo arreglo con los resultados de ejecutar una función para cada elemento del arreglo.
  • filter(): devuelve un nuevo arreglo con los elementos que cumplen con una condición determinada.
  • reduce(): devuelve un valor que es el resultado de ejecutar una función para cada elemento del arreglo.
