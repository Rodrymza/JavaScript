Conceptos importantes
    - Target (blanco): 
    En JavaScript, el elemento target se refiere al objeto que desencadena un evento. Por ejemplo, si se hace clic en un botón, el botón es el target del evento de clic. El elemento target se puede acceder a través de la propiedad 'target' del objeto evento.

- Trigger: 
    En JavaScript, un trigger es un evento que activa una función o una acción específica. Por ejemplo, cuando se hace clic en un botón, el evento de clic puede activar una función que muestra un mensaje o realiza una acción específica. Los triggers se utilizan comúnmente en los eventos de los elementos HTML para ejecutar código cuando se produce un evento específico.

- Event Handler:
    Un event handler (manejador de eventos) es una función que se ejecuta en respuesta a un evento específico, como un clic en un botón o la carga de una página. Los event handlers se utilizan para gestionar los eventos que ocurren en una aplicación web y realizar acciones específicas en respuesta a esos eventos.

Por ejemplo, si se hace clic en un botón, el event handler asociado a ese botón se ejecutará y puede realizar acciones como mostrar un mensaje, enviar una solicitud al servidor, o actualizar el contenido de la página.

En JavaScript, los event handlers se pueden agregar a los elementos HTML utilizando la propiedad "addEventListener" o la propiedad "onclick" (para eventos de clic). Por ejemplo:

```javascript
// Agregar un event handler para el evento de clic en un botón
document.getElementById("miBoton").addEventListener("click", function() {
  alert("¡Hola, mundo!");
});
```

En este ejemplo, el event handler es una función que muestra un mensaje de alerta cuando se hace clic en el botón con el id "miBoton".


    Event listener:
    Un event listener (oyente de eventos) es una función que se ejecuta en respuesta a un evento específico, como un clic en un botón o la carga de una página. Los event listeners se utilizan para gestionar los eventos que ocurren en una aplicación web y realizar acciones específicas en respuesta a esos eventos.

En JavaScript, los event listeners se pueden agregar a los elementos HTML utilizando la propiedad "addEventListener". Por ejemplo:

```javascript
// Agregar un event listener para el evento de clic en un botón
document.getElementById("miBoton").addEventListener("click", function(event) {
  console.log("Se hizo clic en el botón");
  console.log("El target del evento es:", event.target);
});
```

En este ejemplo, el event listener es una función que se ejecuta cuando se hace clic en el botón con el id "miBoton". La función recibe un objeto "event" como parámetro, que contiene información sobre el evento que se produjo, como el target del evento.
